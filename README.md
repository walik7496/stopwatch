# Ultra Modern Stopwatch ⏱️

This project is an ultra-modern **stopwatch** built in **Python** with an iPhone-style interface.

## Features

- 🔹 **Stopwatch with start, stop, reset**  
- 🔹 **Lap functionality with lap list**  
- 🔹 **Dark theme with gradient background**  
- 🔹 **Glassmorphism effect** (frosted glass panel)  
- 🔹 **Circular animated progress**  
- 🔹 **Smooth millisecond display**  
- 🔹 **MP3 sound alert for laps**  

---

## 📦 Installation

1. Install Python (>=3.10)  
2. Download or clone this repository  
3. Install required libraries:

```bash
pip install customtkinter pygame
```

Download an MP3 file for lap sound and place it in the same folder as the Python file:

lap.mp3 — lap sound for the stopwatch

🚀 Running
python ultra_stopwatch.py
🎨 User Interface

Gradient animated background

Frosted glass panel (glassmorphism)

Circular progress animation

Smooth, real-time millisecond updates

Dark mode
